# WMS Analytics - Testing & Visualization

This is a **SEPARATE** folder for analytics testing and visualization purposes.

**⚠️ IMPORTANT: This folder is NOT part of the main warehouse management system project.**

## Purpose
- Test analytics visualizations
- Analyze grain storage patterns
- Customer behavior analysis
- NOT to be integrated with the main project

## Visualizations Required

### 1. Grain Movement Analysis
Bar graph showing:
- X-axis: Grain IN/OUT operations
- Y-axis: Amount of grains (in kg or bags)

### 2. Customer Activity Analysis
Bar graph showing:
- How many customers are currently storing grains
- How many customers have stored grains (historical)
- How many customers sold their grains
- At what price they sold (price analysis)
